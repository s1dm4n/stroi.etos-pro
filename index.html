Что такое ЭТОС?

«ЭТОС» – цифровая экосистема для комплексного управления строительными проектами. Она состоит из функциональных модулей - готовых программ для решения конкретных задач на разных этапах строительства.

Преимущества

Сделано строителями для строителей

Создатели «Этос» – профессиональные строители с опытом работы в АО «Концерн Росэнергоатом», Ростехнадзоре, Мособлгосэкспертизе и других крупнейших организациях отрасли.

Импорто­независимость

Программа для ЭВМ «Этос» внесена в реестр российского ПО (№ 6719), ее безопасность подтверждена сертификатом ФСТЭК по 4-му уровню доверия. Разработчик системы ООО «Стройформ» – аккредитованная IT-компания, имеет лицензии ФСТЭК России и ФСБ.

Кросс­платформенность

Все модули «Этос» – это веб-приложения, для работы которых на стороне клиента нужен только браузер и выход в интернет, а на стороне сервера – Windows или любой из современный Linux-дистрибутивов (Debian, Ubuntu, Astra Linux и пр.).

Постоянное развитие

Благодаря уникальной архитектуре и использованию low-code технологий «Этос» быстро адаптируется под любые изменения законодательства, с каждой новой версией предоставляя пользователям всё новые и новые возможности по управлению проектами.
